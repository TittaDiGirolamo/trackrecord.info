# Trackrecord Design Principles

**Version**: 1.1  
**Last Updated**: 2026-07-03

This folder contains the official design principles that guide the development of Trackrecord.info.

The principles prioritize calm, trustworthy, and evidence-based user experiences while maintaining high standards of transparency and ethical design.

## Main Document

- [DESIGN-PRINCIPLES.md](./DESIGN-PRINCIPLES.md) — Complete set of 7 core principles with rationale and application guidance.

## Key Decisions

- **Typography**: Inter is used as the primary typeface across all interfaces.
- **Approach**: Mobile-first, clarity-focused, and data-driven.
- **Philosophy**: Respect user attention and build long-term credibility.
