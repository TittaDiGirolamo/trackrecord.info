# Trackrecord.info — 90-Day Action Plan (Based on v1.5 Scope)

**Version**: 1.0

**Date**: 2026-06-08

**Aligned to**: Problem & Vision Document v1.5 (Musk-simplified)

**Baseline (2026-06-08)**: 0 sign-ups • Small number of resolved predictions • Basic static site • WC2026 English-language focus (no Oranje restriction) • Human-reviewed resolutions only • Lean MVP (no user accounts or advanced search in first phase)

**Overall 90-Day Goal**

Prove the core **log → resolve (human-reviewed) → score → display** loop with high verification integrity using **FIFA World Cup 2026 predictions from English-language sources**. Achieve measurable data quality and early demand signals with a deliberately minimal feature set.

**Key Targets by End of 90 Days (≈ Sep 6, 2026)**

- 50–70+ resolved WC2026 predictions (English sources) from ≥ 10–12 distinct figures
- Methodology v0.8 + Editorial Charter published
- 150–400 sign-ups
- Clean, updated public table of predictions and accuracy scores
- Clear foundation for advisory board outreach

---

## Phase 1: Days 1–30 — Foundation & Data Momentum

**Dates**: 2026-06-09 → 2026-07-08

**Focus**: Build high-quality WC2026 data with human review and refresh public presence.

### Week 1–2 Priorities

- Expand gold standard to 20–25 high-quality records focused on **FIFA World Cup 2026** (English-language sources: major outlets, analysts, models).
- Resolve all currently pending easy-to-verify WC2026 predictions using official sources + human review.
- Update `predictions.jsonl`, regenerate resolution report, and refresh `index.html` + README with new data.
- Finalize METHODOLOGY.md v0.8.

### Week 3–4 Priorities

- Add 10–15 new resolved WC2026 predictions.
- Improve landing page clarity and basic stats display.
- Set up simple sign-up tracking.
- Publish first public GitHub update with new resolved examples.

**Success Metrics (End of Phase 1)**

- ≥ 25 gold standard WC2026 records
- ≥ 20–25 total resolved predictions
- Site and reports updated
- Methodology v0.8 published

## Resource & Risk Notes

**Assumed Capacity**: 15–25 focused hours/week (solo founder).

**Risks & Mitigations**

- **Risk**: Slower data collection without Oranje-specific network → **Mitigation**: Leverage major English-language sources (ESPN, The Athletic, BBC, Opta, prominent analysts). WC2026 has high global coverage.
- **Risk**: Low sign-up growth → **Mitigation**: Prioritize transparent, high-quality public data and methodology. Credibility drives organic interest.
- **Risk**: Scope creep into user accounts or non-WC2026 topics → **Mitigation**: Strict adherence to v1.5 In/Out of scope.

**Success Definition**

By early September 2026, Trackrecord.info has a working, auditable core loop for **FIFA World Cup 2026 English-language predictions**, published governance documents, and meaningful early demand signals — providing a clean foundation for the next phase.
